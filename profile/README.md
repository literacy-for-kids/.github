# 📚 Literacy for Kids

**Literacy for Kids** is a collection of open-source curricula designed for children ages 8–12.

The project helps children understand the systems that shape modern life — topics that are often missing from traditional education but essential for navigating the modern world.

---

## 🎯 What We Teach

- **Decision Making** — how to think clearly and evaluate choices
- **Technology** — how computers and digital systems work
- **Information** — how to find, read, and evaluate sources
- **Money** — how financial systems affect everyday life
- **Governance** — how societies are organized and how citizens participate

---

## 📖 Curricula

Each curriculum is short, discussion-driven, and designed for parents, educators, and homeschool families. Lessons typically take **10–20 minutes** and are easy to integrate into classrooms, homeschool environments, or after-school programs.

| | Curriculum | Description |
|---|---|---|
| 🧠 | [Decision Literacy for Kids](https://github.com/literacy-for-kids/decision_literacy_for_kids) | How to think clearly, evaluate choices, and understand consequences. |
| 💻 | [Computer Literacy for Kids](https://github.com/literacy-for-kids/computer_literacy_for_kids) | How computers work and how to use technology responsibly. |
| 📰 | [Media Literacy for Kids](https://github.com/literacy-for-kids/media_literacy_for_kids) | How information spreads and how to evaluate sources critically. |
| 💰 | [Financial Literacy for Kids](https://github.com/literacy-for-kids/financial_literacy_for_kids) | How money works and how financial decisions affect the future. |
| 🏛 | [Civic Literacy for Kids](https://github.com/literacy-for-kids/civic_literacy_for_kids) | How societies organize themselves and how citizens shape communities. |

## 💡 Project Philosophy

We focus on **foundational literacies** rather than traditional academic subjects. Each curriculum helps children build mental models of how the world actually works — encouraging:

- Critical thinking
- Understanding systems
- Curiosity and discussion

The goal is not to give children all the answers, but to equip them with the frameworks to ask better questions.

## 🤝 Open Source

All curricula are **free to use, open source, and adaptable**.
